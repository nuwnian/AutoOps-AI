About AutoOps-AI

AutoOps-AI is a minimal web application project designed to showcase cloud deployment, automation, and monitoring skills on Microsoft Azure. It demonstrates real-world DevOps practices, including CI/CD automation, health monitoring, and self-healing scripts for increased reliability. The project is intentionally simple to focus on infrastructure, automation, and best practices.

  • Azure App Service deployment
  • CI/CD with GitHub Actions
  • Health monitoring & self-healing
  • Modern, clean UI
  • Clear documentation
